````markdown
## Viewing the history

```bash
git log --stat        # Shows the list of modified files per commit
git log --patch       # Shows the actual changes (patches)
```
````

## Filtering the history

```bash
git log -3                         # Shows the last 3 commits
git log --author="Mosh"           # Commits by author
git log --before="2020-08-17"     # Commits before date
git log --after="one week ago"    # Commits after relative date
git log --grep="GUI"              # Commits with "GUI" in the message
git log -S"GUI"                   # Commits where the patch contains "GUI"
git log hash1..hash2                # Range of commits between two hashes
git log -- <file.txt>               # Commits that touched file.txt
```

## Formatting the log output

```bash
git log --pretty=format:"%an committed %H"    # Custom format (author and commit hash)
```

## Creating an alias

```bash
git config --global alias.lg "log --oneline"
```

## Viewing a commit

```bash
git show HEAD~2                 # Show the commit two steps before HEAD
git show HEAD~2:file1.txt       # Show file1.txt as stored in that commit
```

## Comparing commits

```bash
git diff HEAD~2 HEAD                 # Changes between two commits
git diff HEAD~2 HEAD -- file.txt     # Changes to file.txt only
```

## Checking out a commit (detached HEAD)

```bash
git checkout dad47ed          # Checks out the given commit (detached HEAD)
git checkout master           # Switches back to the master branch
```

## Finding a bad commit with bisect

```bash
git bisect start
git bisect bad                 # Mark current commit as bad
git bisect good ca49180        # Mark a known-good commit
git bisect reset               # Terminate bisect session
```

## Finding contributors

```bash
git shortlog                   # Summary of contributors and commit counts
```

## Viewing the history of a file

```bash
git log -- <file.txt>          # Commits that touched file.txt
git log --stat -- <file.txt>   # Stats for changes to file.txt
git log --patch -- <file.txt>  # Patches for changes to file.txt
```

## Finding the author of lines

```bash
git blame file.txt             # Shows the author of each line in file.txt
```

## Tagging

```bash
git tag v1.0                   # Tag the last commit as v1.0
git tag v1.0 5e7a828           # Tag an earlier commit
git tag                        # List all tags
git tag -d v1.0                # Delete the given tag
```
